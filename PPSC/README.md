# PPSC (Параллельное программирование суперкомпьютерных систем)

Данная папка содержит задания по курсу параллельного программирования.

## Структура папки

### Задания

- **No.1** - HPL (High-Performance Linpack) benchmark задание
  - Тестирование производительности суперкомпьютеров Top500
  - Файлы конфигурации HPL.dat и Make.Linux_oneAPI
  
- **No.2** - Анализ списка Top500 суперкомпьютеров
  - Работа с данными TOP500_202406.xlsx

- **No.3** - Параллельное программирование с Parallel STL
  - Вычисление суммы массива
  - Использование std::reduce для параллельных вычислений
  - CMake проект с поддержкой Intel TBB

- **No.4** - Программирование с SIMD интринсиками
  - Демонстрация инструкции `_mm_div_pd`
  - Сравнение производительности SIMD и скалярных вычислений
  - Makefile проект

### Презентации

- **PPT** - Презентации курса (PDF файлы 0-4)

## История консолидации

Задания No.3 и No.4 были перенесены из отдельных репозиториев:
- `PPSC-homework-No.3-58` → `PPSC/No.3`
- `PPSC-homework-No.4-58` → `PPSC/No.4`

Все задания теперь объединены в одном репозитории для удобства управления.
